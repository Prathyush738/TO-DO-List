## 📋 Project Description

A full-featured **To-Do List** application built with Python that helps users manage tasks with **priority levels, categories, and due dates**. Data is stored in **JSON format**. Available in both **Desktop GUI (Tkinter)** and **Web App (Flask)** versions.


## ✨ Features

- ➕ Add tasks with **Title, Priority, Category, Due Date & Note**
- ✅ Mark tasks as **complete** with checkbox (strikethrough effect)
- ✏️ Edit any task details
- 🗑️ Delete tasks with confirmation
- 🎯 **Priority levels** — High 🔴, Medium 🟡, Low 🟢
- 📅 **Due date tracking** with overdue alerts in red
- 🔍 **Live search** by title or note
- 📊 Sidebar stats — Total, Pending, Done, Overdue
- 🗂️ Filter by **Status, Priority & Category**
- 💾 Auto-saves all tasks to `tasks.json`
